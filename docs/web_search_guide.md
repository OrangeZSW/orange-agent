# 联网搜索功能使用指南

## 功能概述

Orange Agent 现已支持联网搜索功能，可以让 AI 助手实时获取互联网信息，提供更准确、更及时的回答。

## 功能特性

### 🔍 搜索引擎搜索
- **DuckDuckGo**（推荐）：免费使用，无需 API Key
- **Google**：需要配置 Custom Search API Key
- **Bing**：需要配置 Bing Search API Key

### 📄 网页内容抓取
- 获取网页正文内容
- 自动提取标题和描述
- 清理 HTML 标签和无关内容
- 支持中英文网页

## 使用方法

### 1. 在对话中使用

直接向 AI 助手提问，它会自动判断是否需要联网搜索：

```
用户：帮我搜索一下 Go 语言的最新版本特性
AI：[自动调用 web_search 工具进行搜索]
```

### 2. 指定搜索类型

```
# 搜索引擎搜索
搜索 "人工智能发展趋势"

# 抓取网页内容
抓取 https://golang.org 的内容
```

### 3. 指定搜索引擎

```
用 DuckDuckGo 搜索 "区块链技术"
用 Google 搜索 "机器学习教程"
```

## 工具参数详解

### web_search 工具

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| query | string | ✅ | 搜索关键词或URL |
| search_type | string | ✅ | 搜索类型：`search` 或 `fetch` |
| engine | string | ❌ | 搜索引擎：`duckduckgo`（默认）、`google`、`bing` |
| num_results | integer | ❌ | 返回结果数量（默认5，最多10） |

### 搜索类型说明

#### search - 搜索引擎搜索
使用搜索引擎搜索关键词，返回相关网页列表。

**示例：**
```json
{
  "query": "Go语言并发编程",
  "search_type": "search",
  "engine": "duckduckgo",
  "num_results": 5
}
```

**返回格式：**
```
🔍 搜索结果 (共 5 条):

1. **Go语言并发编程 - 简书**
   📎 https://www.jianshu.com/p/xxx
   📝 Go语言并发编程的详细介绍...

2. **Go并发模式 - 博客园**
   📎 https://www.cnblogs.com/xxx
   📝 深入理解Go的并发模型...
```

#### fetch - 抓取网页内容
抓取指定URL的网页内容，提取正文。

**示例：**
```json
{
  "query": "https://golang.org/doc/tutorial/",
  "search_type": "fetch"
}
```

**返回格式：**
```
📄 网页内容:

📌 标题: Go Tutorial
📎 URL: https://golang.org/doc/tutorial/
📝 描述: Go语言官方教程

📖 正文内容:
This tutorial introduces the basics of writing Go code...
```

## 最佳实践

### 1. 选择合适的搜索引擎
- **日常搜索**：使用 DuckDuckGo（免费、隐私友好）
- **学术搜索**：使用 Google Scholar
- **中文搜索**：DuckDuckGo 对中文支持良好

### 2. 优化搜索关键词
- 使用具体、明确的关键词
- 避免过于宽泛的搜索词
- 可以使用引号搜索精确短语

### 3. 合理设置结果数量
- 一般搜索：3-5 条结果足够
- 深度研究：可以设置 10 条
- 快速查询：2-3 条即可

### 4. 网页抓取技巧
- 确保URL完整（包含 https://）
- 优先抓取内容型网页
- 避免抓取需要登录的页面

## 注意事项

### 1. 网络要求
- 需要稳定的网络连接
- 某些地区可能需要代理
- DuckDuckGo API 可能在某些地区访问受限

### 2. 内容限制
- 网页内容会截取前 3000 字符
- 某些网站可能阻止爬虫
- 动态加载的内容可能无法获取

### 3. 使用频率
- 建议合理使用，避免频繁请求
- DuckDuckGo 有请求频率限制
- 尊重网站的 robots.txt

### 4. 隐私安全
- 搜索内容会发送到搜索引擎
- 不要搜索敏感个人信息
- 注意保护隐私

## 错误处理

### 常见错误

#### 1. 网络连接失败
```
错误：搜索失败: Get "https://api.duckduckgo.com/...": dial tcp: lookup api.duckduckgo.com: no such host
解决：检查网络连接，或配置代理
```

#### 2. URL 无效
```
错误：无效的 URL: missing protocol scheme
解决：确保URL包含 http:// 或 https://
```

#### 3. 网页无法访问
```
错误：HTTP 错误: 403 Forbidden
解决：该网站禁止爬虫访问，尝试其他来源
```

## 技术实现

### 架构设计
```
用户请求 → AI 判断 → 调用 web_search 工具
                              ↓
                    选择搜索引擎/抓取模式
                              ↓
                    发送 HTTP 请求
                              ↓
                    解析响应内容
                              ↓
                    格式化返回结果
```
